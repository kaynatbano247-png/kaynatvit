NUMBER GUESSING GAME
OVERVIEW-
It is a simple game where a player tries to guess a randomly generated number within a set range, with the computer providing feedback to the player on each guess. After each incorrect guess, the computer tells the player if their guess was "too high","too low",or "correct", guiding them toward the secret number.
FEATURES-
It gives you an option like you want to guess one digit , two digit , three digit or four digit number.Also you certain amount of chances to guess the number it increases with respact to the number of digits increases.you have an option to end or continue the game you can play with it as much as you want.
TECHNOLOGIES/TOOLS USED-
For code i use Pycharm to do python coding.Random module is used and loop like while and for is used .If else statement plays a major role in this code.
you'll have to just execute the code .It gives you instructions how to play.
